# Prompt 2 - Tema Shopify personalizado marca minimalista

**Stack:** Liquid, HTML, CSS, JavaScript, Shopify CLI

---

Crea un tema Shopify personalizado usando Shopify CLI y el ambiente de desarrollo real. Todo lo de la version A mas: 1) configurar con Shopify Partner account (gratuita) y tienda de desarrollo, 2) usar Shopify CLI para desarrollo local con hot reload, 3) configurar Shopify Payments en modo test, 4) implementar metafields para datos adicionales de producto (material, origen, cuidados), 5) seccion de reviews usando metaobjects de Shopify (sin app externa), 6) configurar dominio de prueba. README con paso a paso: crear Partner account, crear dev store, instalar Shopify CLI, clonar tema Dawn, aplicar customizaciones, preview y deploy.
